# ☁️ Cloud Panel v1.9.0 — "Teams & Power-Ups"

> Our biggest update yet — **10 major features**. Update with `sudo cloud-panel-update`.

**👥 Subusers / per-server sharing**
Invite others to a server with **granular permissions** — console, commands, power, files, backups, schedules, databases, players & more. Enforced everywhere (REST **and** the live console).

**⏰ Scheduled tasks (cron)**
Nightly restarts, timed backups and automated commands on a simple cron schedule.

**🔐 Two-factor authentication (TOTP)**
Authenticator-app 2FA with QR enrollment + one-time recovery codes. Optional, and your secret never leaves the server.

**🗄️ Per-server databases**
Provision real **MySQL / MariaDB** databases per server against admin-managed hosts (create, rotate password, delete).

**🧩 Plugin / mod browser (Modrinth)**
Search Modrinth and **one-click install** plugins & mods straight into `plugins/`/`mods/`.

**📦 One-click Modrinth modpacks**
Pick a pack and Cloud Panel installs every mod, override **and** the matching loader automatically.

**🟢 Live player list**
See who's online (parsed live from the console) and **kick/ban** with one click.

**🥚 8 new eggs (19 → 27)**
**Quilt**, **Pufferfish**, **Leaf**, **PocketMine-MP**, and **SteamCMD** games — **Rust**, **Valheim**, **Counter-Strike 2**.

**📊 Historical metrics**
Real CPU / RAM / disk + uptime graphs (1h–7d), not just live sparklines.

**🌐 Public status pages**
A shareable, login-free status page per server at `/status/<slug>` — live status, players & uptime.

**🤖 Bot:** new `/changelog` command (now **27** slash commands).
